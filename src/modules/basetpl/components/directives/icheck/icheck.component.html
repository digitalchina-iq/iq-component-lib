概述

icheck美化单选框，复选框样式

icheck

icheck

<input type="checkbox" name="" icheck> <input type="radio" name="" icheck>

indeterminate

[indeterminate] = "boolean"; 半选状态，boolean为true时半选

<input type="checkbox" name="" icheck [indeterminate]="true">

disabled

[disabled] = "boolean"; 禁用状态，boolean为true时禁用

<input type="checkbox" name="" icheck [disabled]="true"> <input type="radio" name="" icheck [disabled]="true">

allcheck, allcheckChild

全选，由一个icheck，通过allcheck和allcheckChild属性，关联其他多个icheck。控制全选、全不选、半选状态

<div> <input type="checkbox" icheck allcheck="demo" name=""> <ul> <li> <input type="checkbox" icheck allcheckChild="demo" allcheck="demo1" name=""> <ul> <li><input type="checkbox" icheck allcheckChild="demo1" name=""></li> <li><input type="checkbox" icheck allcheckChild="demo1" name=""></li> <li><input type="checkbox" icheck allcheckChild="demo1" name=""></li> </ul> </li> <input type="checkbox" icheck allcheckChild="demo" allcheck="demo2" name=""> <li> <ul> <li><input type="checkbox" icheck allcheckChild="demo2" name=""></li> <li><input type="checkbox" icheck allcheckChild="demo2" name=""></li> <li><input type="checkbox" icheck allcheckChild="demo2" name=""></li> </ul> </li> <input type="checkbox" icheck allcheckChild="demo" allcheck="demo3" name=""> <li> <ul> <li><input type="checkbox" icheck allcheckChild="demo3" name=""></li> <li><input type="checkbox" icheck allcheckChild="demo3" name=""></li> <li><input type="checkbox" icheck allcheckChild="demo3" name=""></li> </ul> </li> </ul> </div>

onClick

(onClick) = "doSomething($event)"; 单选框复选框，点击事件，$event为单选框复选框的value值

<input type="checkbox" name="" icheck (onClick) = "doSomething($event)";>

onChange

(onChange) = "doSomething($event)"; 单选框复选框，值改变事件，$event为单选框复选框的value值

<input type="checkbox" name="" icheck (onChange) = "doSomething($event)";>

onIndeterminate

(onIndeterminate) = "doSomething($event)"; 单选框复选框，半选状态改变事件，$event为半选状态布尔值

<input type="checkbox" name="" icheck (onIndeterminate) = "doSomething($event)";>

onCount

(onCount) = "doSomething($event)"; 单选框复选框，子项选中计数，$event为子项选中的个数

<input type="checkbox" name="" icheck (onCount) = "doSomething($event)";>
参数 说明 类型 可选值 默认值 是否必选
