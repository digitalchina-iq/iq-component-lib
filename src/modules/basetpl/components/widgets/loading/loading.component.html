概述

加载中组件，显示加载中效果

show

show:boolean； 如果为true显示加载中效果，如果为false组件消失

<loading [show]="true"></loading>

type

type:string； 加载效果类型，目前六种：circle1,circle2,square1,round1,round2,various1，默认为circle1

如果设置[type]="'none'"，可以在组件里自定义加载中效果

  • circle1
  • circle2
  • square1
  • round1
  • round2
  • various1
  • none

    loading...
<loading [show]="true" type="circle1"></loading> <loading [show]="true" type="circle2"></loading> <loading [show]="true" type="square1"></loading> <loading [show]="true" type="round1"></loading> <loading [show]="true" type="round2"></loading> <loading [show]="true" type="various1"></loading> <loading [show]="true" type="none"> <p style="margin: 0;color: red;">loading...</p> </loading>

style

style:any； 定义组件的额外样式

<loading [show]="true" [style]="{{'{'}}background:'red'{{'}'}}"></loading>
参数 说明 类型 可选值 默认值 是否必选
