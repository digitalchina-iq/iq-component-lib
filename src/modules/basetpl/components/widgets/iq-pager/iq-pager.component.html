概述

分页器组件

data

data:Pager<{{'{'}}pageSize:number,pageNo:number,total:number,totalPages:number{{'}'}}>；data值为Pager对象，pageSize:每页数量，pageNo:页码，total:总数，totalPages:总页数

<iq-pager [data]="pagerData"></iq-pager> pagerData = new Pager().set({{'{'}}pageSize:10,pageNo:2,total:123,totalPages:12{{'}'}})

onChange

切换分页事件，参数$event为新的Pager对象

页面大小：{{pagerData.pageSize}}，页码：{{pagerData.pageNo}}

<iq-pager [data]="pagerData" (onChange)="pageChange($event)"></iq-pager>
参数 说明 类型 可选值 默认值 是否必选
