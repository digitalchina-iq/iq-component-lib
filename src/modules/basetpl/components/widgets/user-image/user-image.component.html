概述

显示人员信息的组件。

user

user: Person|string; Person对象或者人员id

<div user-image [user]="someone" ></div>

hasClose

hasClose: boolean; 是否能使用删除操作，配合onClose事件删除人员

<div user-image [user]="someone" [hasClose]="true" ></div>

hasImg

hasImg: boolean; 是否显示人员头像，默认显示

<div user-image [user]="someone" [hasImg]="false"></div>

showHover

showHover: boolean; 是否显示hover详细信息

<div user-image [user]="someone" [showHover]="true" ></div>

onClose

onClose，点击删除触发的事件

<div user-image [user]="someone" [hasClose]="true" (onClose)="deleteSomeOne()" ></div>
参数 说明 类型 可选值 默认值 是否必选
