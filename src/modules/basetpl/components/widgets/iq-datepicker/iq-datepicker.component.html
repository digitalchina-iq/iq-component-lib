概述

选择或输入日期，支持年、月、日期等类型，支持选择范围。

required

required:boolean； 如果为true，则日期必选，不选日期组件边框会变红

<iq-datepicker [(ngModel)]="date" [required]="true" ></iq-datepicker>

disabled

disabled:boolean； 如果为true，则组件被禁用

<iq-datepicker [(ngModel)]='date' disabled></iq-datepicker>

minYear, maxYear

minYear:number, maxYear:number； minYear为组件可选用的最小年份，maxYear为组件可选用的最大年份

<iq-datepicker [(ngModel)]="date" minYear="2000" maxYear="2020"></iq-datepicker>

placeHolder

placeHolder:string； 设置组件的默认提示文字

<iq-datepicker [(ngModel)]="date" placeHolder="-请选择日期"></iq-datepicker>

startDate, endDate

startDate:Date|string|number, endDate:Date|string|number； 设置组件的可选择的起始日期和结束日期

~
<iq-datepicker [(ngModel)]="startDate" [endDate]="endDate"></iq-datepicker> <iq-datepicker [(ngModel)]="endDate" [startDate]="startDate"></iq-datepicker>

format

format:string； format参考moment.js的format格式，设置组件的返回值的格式

返回值：{{date[6]}}

<iq-datepicker [(ngModel)]="date" format="YYYY-MM-DD"></iq-datepicker>
参数 说明 类型 可选值 默认值 是否必选
required 是否必选 boolean true,false false 否
disabled 是否禁用 boolean true,false 否
minYear 最小年份 string,number 当年年份-70 否
maxYear 最大年份 string,number 当年年份+30 否
placeHolder 设置默认文字 string 否
startDate 开始日期 string,date,number 否
endDate 结束日期 string,date,number 否
format 格式化（YYYY-MM-DD） string 否

简单示例：选择日期区间

~ 
	    		
<div style="display: inline-block;width: 250px;">
 <iq-datepicker [(ngModel)]="startDate" [endDate]="endDate" placeHolder="-请选择开始日期"></iq-datepicker>
</div>
~
<div style="display: inline-block;width: 250px;">
 <iq-datepicker [(ngModel)]="endDate" [startDate]="startDate" placeHolder="-请选择结束日期"></iq-datepicker>
</div>
	        
	    	
	    		
startDate: Date;
endDate: Date;

必选校验

 
	    		
<iq-datepicker [(ngModel)]="someDate" [required]="isSubmit" placeHolder="-请选择日期"></iq-datepicker>
<button class="m-btn-success" (click)="submitSomething()">提交</button>
<button class="m-btn-cancel" (click)="reset()">重置</button>
	        
	    	
	    		
someDate: Date;
submitSomething() {{'{'}}
 this.isSubmit = true;
{{'}'}}
reset() {{'{'}}
 this.someDate = undefined;
 this.isSubmit = false;
{{'}'}}
