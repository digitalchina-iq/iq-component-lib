概述

tab切换组件

tabIndex, tabContent

tabIndex: string, tabContent: string； 定义组件的tab和content的对应关系

  • abc
  • def
  • ghi
abc
def
ghi
<iq-tab> <ul class="tab-index clearfix"> <li tabIndex="abc">abc</li> <li tabIndex="def">def</li> <li tabIndex="ghi">ghi</li> </ul> <div class="tab-cont"> <div tabContent="abc">abc</div> <div tabContent="def">def</div> <div tabContent="ghi">ghi</div> </div> </iq-tab>

tabActiveClass

tabActiveClass: string； 定义组件的tab激活时的className

  • abc
  • def
  • ghi
abc
def
ghi
<iq-tab tabActiveClass="active"> <ul class="tab-index clearfix"> <li tabIndex="abc">abc</li> <li tabIndex="def">def</li> <li tabIndex="ghi">ghi</li> </ul> <div class="tab-cont"> <div tabContent="abc">abc</div> <div tabContent="def">def</div> <div tabContent="ghi">ghi</div> </div> </iq-tab>

tabActiveIndex

tabActiveIndex: string； 定义组件的被默认激活的tab

  • abc
  • def
  • ghi
abc
def
ghi
<iq-tab tabActiveClass="active" tabActiveIndex="def"> <ul class="tab-index clearfix"> <li tabIndex="abc">abc</li> <li tabIndex="def">def</li> <li tabIndex="ghi">ghi</li> </ul> <div class="tab-cont"> <div tabContent="abc">abc</div> <div tabContent="def">def</div> <div tabContent="ghi">ghi</div> </div> </iq-tab>

onTab

(onTab)="doSomething($event)"；tab切换事件，$event为tab的Id

  • abc
  • def
  • ghi
abc
def
ghi
<iq-tab tabActiveClass="active" (onTab)="print($event)"> <ul class="tab-index clearfix"> <li tabIndex="abc">abc</li> <li tabIndex="def">def</li> <li tabIndex="ghi">ghi</li> </ul> <div class="tab-cont"> <div tabContent="abc">abc</div> <div tabContent="def">def</div> <div tabContent="ghi">ghi</div> </div> </iq-tab>
参数 说明 类型 可选值 默认值 是否必选
tabIndex 选项卡Id string 是
tabContent 选项卡对应内容Id string 是
tabActiveClass 选项卡被激活时添加的className string 否
tabActiveIndex 选项卡默认被激活的项 string 否

简单示例：增加或删除tab选项

  • {{item.name}}
{{item.content}}
 
          
<button class="m-btn-success" (click)="addItem()">增加一项</button>
<div class="tab-demo">
  <iq-tab tabActiveClass="active">
    <ul class="tab-index">
      <li *ngFor="let item of tabList;index as i" [tabIndex]="item.id">{{'{'}}{{'{'}}item.name{{'}'}}{{'}'}}<i class="iqon-close" [hidden]="tabList.length === 1" (click)="deleteItem(i)"></i></li>
    </ul>
    <div class="tab-cont">
      <div *ngFor="let item of contList" [tabContent]="item.id">{{'{'}}{{'{'}}item.content{{'}'}}{{'}'}}</div>
    </div>
  </iq-tab>
</div>
          
        
          
tabList: {{'{'}}id:string,name: string{{'}'}}[] = [{{'{'}}name:'abc',id: 'abc'{{'}'}},{{'{'}}name:'def',id: 'def'{{'}'}},{{'{'}}name:'ghi',id: 'ghi'{{'}'}}];
contList: {{'{'}}id:string,content: string{{'}'}}[] = [{{'{'}}content:'abc',id: 'abc'{{'}'}},{{'{'}}content:'def',id: 'def'{{'}'}},{{'{'}}content:'ghi',id: 'ghi'{{'}'}}];
count: number = 0;

deleteItem(i) {{'{'}}
  this.tabList.splice(i,1);
  this.contList.splice(i,1);
{{'}'}}

addItem() {{'{'}}
  this.count++;
  this.tabList.push({{'{'}}name: `newTab${{'{'}}this.count{{'}'}}`, id: `newId${{'{'}}this.count{{'}'}}`{{'}'}});
  this.contList.push({{'{'}}content: `newContent${{'{'}}this.count{{'}'}}`, id: `newId${{'{'}}this.count{{'}'}}`{{'}'}});
{{'}'}}
