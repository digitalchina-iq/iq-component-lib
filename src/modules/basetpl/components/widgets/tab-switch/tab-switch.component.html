概述

tab切换组件

tabIndex, tabContent

tabIndex: string, tabContent: string； 定义组件的tab和content的对应关系

  • abc
  • def
  • ghi
abc
def
ghi
<iq-tab> <ul class="tab-index clearfix"> <li tabIndex="abc">abc</li> <li tabIndex="def">def</li> <li tabIndex="ghi">ghi</li> </ul> <div class="tab-cont"> <div tabContent="abc">abc</div> <div tabContent="def">def</div> <div tabContent="ghi">ghi</div> </div> </iq-tab>

tabActiveClass

tabActiveClass: string； 定义组件的tab激活时的className

  • abc
  • def
  • ghi
abc
def
ghi
<iq-tab tabActiveClass="active"> <ul class="tab-index clearfix"> <li tabIndex="abc">abc</li> <li tabIndex="def">def</li> <li tabIndex="ghi">ghi</li> </ul> <div class="tab-cont"> <div tabContent="abc">abc</div> <div tabContent="def">def</div> <div tabContent="ghi">ghi</div> </div> </iq-tab>

tabActiveIndex

tabActiveIndex: string； 定义组件的被默认激活的tab

  • abc
  • def
  • ghi
abc
def
ghi
<iq-tab tabActiveClass="active" tabActiveIndex="def"> <ul class="tab-index clearfix"> <li tabIndex="abc">abc</li> <li tabIndex="def">def</li> <li tabIndex="ghi">ghi</li> </ul> <div class="tab-cont"> <div tabContent="abc">abc</div> <div tabContent="def">def</div> <div tabContent="ghi">ghi</div> </div> </iq-tab>
参数 说明 类型 可选值 默认值 是否必选
tabIndex 选项卡Id string 是
tabContent 选项卡对应内容Id string 是
tabActiveClass 选项卡被激活时添加的className string 否
tabActiveIndex 选项卡默认被激活的项 string 否
