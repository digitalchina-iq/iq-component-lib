概述

选择时间，选择时间范围

required

required:boolean； 如果为true，则时间必选，不选时间组件边框会变红

<iq-timepicker [(ngModel)]="time" [required]="true" ></iq-timepicker>

disabled

disabled:boolean； 如果为true，则组件被禁用

<iq-timepicker [(ngModel)]='time' disabled></iq-timepicker>

startTime, endTime

startTime:Date|string|number, endTime:Date|string|number； startTime为组件可选用的开始时间，endTime为组件可选用的结束时间

~
<iq-timepicker [(ngModel)]="time" startTime="10:00:00" endTime="20:00:00"></iq-timepicker> <div style="display: inline-block;width: 45%;"> <iq-timepicker [(ngModel)]='startTime' [endTime]="endTime"></iq-timepicker> </div> ~ <div style="display: inline-block;width: 45%;"> <iq-timepicker [(ngModel)]='endTime' [startTime]="startTime"></iq-timepicker> </div>

placeHolder

placeHolder:string； 设置组件的默认提示文字

<iq-timepicker [(ngModel)]="time" placeHolder="-请选择时间"></iq-timepicker>

defaultTime

defaultTime:string； 设置组件下拉项的默认时间

<iq-timepicker [(ngModel)]="time" defaultTime="10:10:10"></iq-timepicker>

format

format:string； format参考moment.js的format格式，设置组件的返回值的格式

返回值：{{time[6]}}

<iq-timepicker [(ngModel)]="time" format="HH:mm:ss"></iq-timepicker>
参数 说明 类型 可选值 默认值 是否必选
required 是否必选 boolean true,false false 否
disabled 是否禁用 boolean true,false 否
minYear 最小年份 string,number 当年年份-70 否
maxYear 最大年份 string,number 当年年份+30 否
placeHolder 设置默认文字 string 否
startDate 开始日期 string,date,number 否
endDate 结束日期 string,date,number 否
format 格式化（YYYY-MM-DD） string 否
