概述

弹出提示警告框:alert

  1. title标题
  2. message内容
  3. type类型---（success，fail，warn）

弹出提示警告框:confirm

  1. message内容

何时使用

需要明确的给出提示信息时使用

Alert

弹出的Alert框

this.windowservice.alert({{'{'}}message:"填写的信息",title:"标题",type:"fail"{{'}'}})

Confirm

弹出的Confirm框

this.windowservice.confirm({{'{'}}message:"填写的信息"{{'}'}})

prompt

弹出的prompt框

this.windowservice.prompt({{'{'}}message:"填写的信息"{{'}'}})
参数 说明 类型 可选值 默认值 是否必选

Alert简单示例

 
	    		
						this.windowservice.alert({{'{'}}message:"填写的信息",title:"标题",type:"success"{{'}'}})
						this.windowservice.alert({{'{'}}message:"填写的信息",title:"标题",type:"fail"{{'}'}})
						this.windowservice.alert({{'{'}}message:"填写的信息",title:"标题",type:"warn"{{'}'}})
						this.windowservice.alert({{'{'}}message:"填写的信息",title:"标题",type:"warn"{{'}'}}, {{'{'}}autoClose: false{{'}'}})
						this.windowservice.alert({{'{'}}message:"填写的信息",title:"标题",type:"warn"{{'}'}}, {{'{'}}closeTime: 1000{{'}'}})
						this.windowservice.alert({{'{'}}message:"#{{'{'}}5{{'}'}}秒后关闭",title:"标题",type:"success"{{'}'}})

Confirm简单示例

 
	    		
						this.windowservice.confirm({{'{'}}message:"填写的信息"{{'}'}})

prompt简单示例

 
	    		
						this.windowservice.prompt({{'{'}}message:"填写的信息"{{'}'}})
